📰 Stylish News Article Website

A modern and responsive News Article Web Page built using HTML5 and CSS3. This project features a professional news layout with well-structured content, attractive typography, working navigation links, and a clean reading experience designed to resemble real-world online news platforms.

Professional news article layout
Fully functional navigation links
Modern and responsive design
Clean typography for better readability
Styled sections and article cards
Mobile-friendly user interface
Built entirely with HTML and CSS

HTML5
CSS3
